# Changelog

All notable changes to this project will be documented in this file.

## [0.0.1] - 2026-03-28
### Added
- Initial release of **Laravel Playground** extension.
- Ability to **open Laravel Playground** using the command `laravel-playground.run`.
- Ability to **run Laravel code** using the command `laravel-playground.execute`.
- Added **editor menu integration** for running code when `laravelPlaygroundActive`.
- Included extension icon (`images/icon.png`) and metadata in `package.json`.
